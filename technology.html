Technology Stack

Technical Platform & Tooling

A comprehensive register of the spatial platforms, databases, analytical tools, and programming environments that underpin Airfree Geospatial's enterprise service delivery. Every platform listed is validated for production deployment in government and enterprise contexts.

Category 01

Spatial Databases

The foundation of enterprise spatial systems. Our spatial database practice is built on open standards and production-tested architectures capable of managing large-scale geospatial datasets with full topological integrity.

PostGIS
Primary Spatial Database

Spatial extension for PostgreSQL. Used for all enterprise spatial database deployments. Supports geometry, raster, topology, and routing functions. OGC-compliant WKT/WKB geometry storage, GIST indexing, and full SQL/MM spatial operations.

PostgreSQL
Enterprise Database Engine

Production-grade relational database backing all spatial data infrastructure deployments. Configured for high-availability, replication, connection pooling (PgBouncer), and enterprise security.

SpatiaLite
Embedded Spatial Database

SQLite with spatial extensions for field data collection, offline GIS operations, and lightweight deployments. Used in mobile and disconnected geospatial workflows.

Oracle Spatial
Enterprise Oracle Integration

Spatial data integration and migration support for organisations operating Oracle environments. SDO_GEOMETRY interoperability, schema migration, and ETL between Oracle Spatial and PostGIS.

Category 02

GIS Platforms

Desktop and server GIS platforms for data creation, analysis, visualisation, and service publishing. We work across both open-source and proprietary platforms to match the institutional requirements of each client.

QGIS
Open-Source Desktop GIS

Primary desktop GIS for spatial analysis, cartographic production, and plugin-based workflow automation. PyQGIS scripting for batch processing and custom tool development.

ArcGIS Pro
Esri Desktop Platform

Advanced spatial analysis, geodatabase management, and model building for clients operating within Esri enterprise environments. ArcPy scripting and ModelBuilder workflows.

ArcGIS Online
Cloud GIS Portal

Configuration, administration, and content management for ArcGIS Online and ArcGIS Enterprise Portal deployments. Web app development, Dashboard creation, and ArcGIS Experience Builder.

GeoServer
OGC Service Server

Open-source geospatial data server for publishing WMS, WFS, WCS, and WPS services. Used in all enterprise SDI deployments for OGC-compliant spatial data distribution.

MapServer
Cartographic Tile Server

High-performance web map service for raster and vector tile delivery. Deployed in government SDI environments requiring OGC WMS/WFS compliance with high-volume request handling.

GeoNode
Spatial Data Catalogue

Open-source geospatial CMS for spatial data discovery, metadata management, and dataset sharing. Used to build INSPIRE-compliant spatial data catalogues for government SDI implementations.

Category 03

Remote Sensing Platforms

Multi-source satellite and airborne imagery platforms, analysis tools, and cloud processing environments for earth observation analytics at any scale.

Sentinel-2
ESA — 10m Multispectral

Primary multispectral satellite for vegetation analysis, land cover classification, and change detection. 13-band imagery at 10–60m resolution with 5-day revisit cycle.

Landsat-9
USGS — Long Archive

50-year Landsat archive used for long-term change detection and historical land cover analysis. Essential for decadal trend analysis and pre-development baseline assessments.

Google Earth Engine
Cloud Remote Sensing

Cloud-based planetary-scale analysis using the complete Sentinel, Landsat, and MODIS archives. Used for large-area classification, time-series compositing, and change detection at regional to national scale.

SNAP
ESA Toolbox

Sentinel Application Platform for SAR processing, radiometric correction, atmospheric correction, and InSAR analysis. Used for surface deformation and subsidence monitoring applications.

ENVI / IDL
Hyperspectral Analysis

Advanced hyperspectral and multispectral image processing for mineral mapping, vegetation stress analysis, and surface material classification using airborne and satellite sensor data.

MODIS / VIIRS
Regional Monitoring

Coarse-resolution daily products for fire detection (MOD14), sea surface temperature, and continental-scale vegetation monitoring. Used in conjunction with Sentinel-2 for multi-scale analysis.

Category 04

UAV & Photogrammetry

End-to-end UAV survey and photogrammetric processing platforms for high-resolution spatial data acquisition and 3D modelling.

Agisoft Metashape
Photogrammetric Processing

Professional photogrammetric processing software for dense point cloud generation, 3D mesh modelling, DSM and DTM production, and orthomosaic generation from UAV and aerial imagery.

Pix4D
UAV Mapping Suite

Cloud-enabled photogrammetric processing and flight planning suite. Pix4Dmapper for photogrammetry, Pix4Dfields for precision agriculture, and Pix4Dinspect for asset inspection workflows.

CloudCompare
Point Cloud Analysis

Open-source point cloud processing and analysis. Ground filtering, classification, volumetric analysis, change detection via cloud-to-cloud distance, and LiDAR data QC workflows.

DJI Terra
Flight Planning & Processing

DJI's enterprise flight planning and photogrammetric processing platform. Mission planning, RTK/PPK workflow integration, and 2D/3D reconstruction for DJI Enterprise UAV platforms.

LAStools
LiDAR Processing Suite

LiDAR data processing, filtering, classification, and tiling. LAZ/LAS format handling, ground point extraction, canopy height model generation, and topographic derivative production.

Open Drone Map
Open-Source Processing

Open-source photogrammetric pipeline for UAV imagery processing. Used in self-hosted and air-gapped environments where cloud processing of sensitive aerial data is not appropriate.
